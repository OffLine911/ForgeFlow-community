# Getting Started with Templates

## What is a Template?

A template is a pre-built workflow that users can import with one click. Templates save time by providing ready-to-use automation patterns.

## Using Community Templates

1. Open ForgeFlow
2. Click **Templates** in the header
3. Switch to the **Community** tab
4. Browse available templates
5. Click a template to select it
6. Click **"Use Template"** to import

## Creating Your Own

To create a template:

1. Build your workflow in ForgeFlow
2. Export it via **Import/Export**
3. Convert to template format (see [Template Structure](structure.md))
4. Submit to the community repo
